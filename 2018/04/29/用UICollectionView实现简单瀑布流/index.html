认识UICollectionView

什么是UICollectionView

简单的UICollectionView就是一个GridView，以多列的方式来呈现数据。一个标准的UICollectionView包含三个部分，它们都是UIView的子类。

  1. Cell: 用于展示内容的主体，对于不同的Cell，可以指定不同的尺寸大小和不同的内容。
  2. Supplementary Views: 在UiCollectionView的Section中，我们可以为其增加Header View和Footer View，它是一个可重用的视图（UICollectionReusableView），可以创建UICollectionReusableView的两个子类，一个是Header View，一个是Footer View。
  3. Decoration Views: “装饰”视图，这是每个Section的背景。

UICollectionView视图层级

UICollectionView有4层视图，从下向上依次为：

UICollectionView：作为自身的容器
backgroundView：用于Cell的默认背景
selectedBackgroundView：Cell被选择是的背景
contentView：自定义内容

UICollectionLayout

UICollectionView并不像UITableView一样是系统自动布局好的，是需要我们自己定义布局的。UICollectionView有个强大的布局引擎——UICollectionViewLayout，它将负责各个Cell，Supplementary View和Decoration View进行组织和调配，设置位置、尺寸、透明度等属性（不过也是有坑的，这个坑在实现瀑布流的时候告诉大家）。

如果不需要自己定制UICollectionViewLayout子类，可以直接使用SDK直接提供的UICollectionViewFlowLayout。FlowLayout是一种直线对齐的布局，就是从第一行从左到右然后从第二行从左到右……，对FlowLayout，你可以设置Cell的大小（是对所有Cell设置，单独设置需要自己制定Layout）和间隔。

实现一个简单瀑布流

创建自定义Layout

代码中定义了一个MyLayoutDelegate，ViewController通过代理的sizeForItemAtIndexPath方法告知Layout，Layout再将每个Cell的属性存放在一个数组中，最终返回一个数组对象设置Cell布局。

上面的代码主要是VC获得Cell的size，在对Cell进行属性设置并返回，这其中有一个坑就是，自己定义Layout使得系统不知道contentSize是多少了，便默认使contentSize等于frameSize了，因此调试时collectionView是不能滚的的，要想使collection依旧能够滚动，就需要重写下面这个方法返回contentSize。

VC部分实现

//  ViewController.m
//  CollectionViewDemo
#import "ViewController.h"
#import "MyLayout.h"
@interface ViewController ()
@end

@implementation ViewController
- (void)viewDidLoad {
    [super viewDidLoad];
    MyLayout *layout = [[MyLayout alloc]init];
    ......
    UICollectionView *collection = [[UICollectionView alloc]initWithFrame: self.view.bounds collectionViewLayout:layout];
    ......
    [collection registerClass:[UICollectionViewCell class] forCellWithReuseIdentifier:@"cellid"];     //注册cell
    [self.view addSubview:collection];
}
- (UICollectionViewCell *)collectionView:(UICollectionView *)collectionView cellForItemAtIndexPath:(NSIndexPath *)indexPath{
    UICollectionViewCell *cell = [collectionView dequeueReusableCellWithReuseIdentifier:@"cellid" forIndexPath:indexPath];
    cell.backgroundColor = [UIColor colorWithRed:(arc4random()%255)/255.0 green:(arc4random()%255)/255.0 blue:(arc4random()%255)/255.0 alpha:1];     //随机背景颜色
    return cell;
}
- (CGSize)sizeForItemAtIndexPath:(NSIndexPath *)indexPath{   //代理方法，告诉layout尺寸
    float WIDTH = ITEM_WIDTH;
    return CGSizeMake(WIDTH, arc4random()%50+50);    //高度在50-100之间
}
